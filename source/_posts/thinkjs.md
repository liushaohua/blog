ThinkJS基于`MVC`设计，把数据视图表现分离得当，把常用的Node方法进行了封装，入门简单，大大的提高了开发效率，让我们可以随随便便分分钟构建一个web应用，废话不多说，直接来。

### 安装ThinkJS
- npm install thinkjs@2 -g --verbos

### 创建项目&&启动
- thinkjs new project_path --es6
- cd project_path && npm install
- npm start
